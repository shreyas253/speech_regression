gmmb_normalize

PURPOSE

GMMB_NORMALIZE Normalize a matrix so that row sums are one.

SYNOPSIS

function p_out = gmmb_normalize(p_in);

DESCRIPTION 

GMMB_NORMALIZE   Normalize a matrix so that row sums are one.

     p_out = GMMB_NORMALIZE(p_in)

     p_in = N x K matrix
     p_out = N x K matrix

  If an unnormalized row sum would be zero,
  the row is left untouched.
